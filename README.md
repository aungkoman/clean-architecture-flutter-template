### Clean Architecture in Flutter

Project structure က ဒါမျိုးရှိမယ်။
```
├───lib
│   ├───core
│   └───features
│       └───one
│           ├───data
│           │   ├───datasources
│           │   ├───models
│           │   └───repositories
│           ├───domain
│           │   ├───entities
│           │   ├───repositories
│           │   └───usecases
│           └───presentation
│               ├───pages
│               ├───providers
│               ├───routes
│               └───widgets
├───test
└───web
    └───icons
```
